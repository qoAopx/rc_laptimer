# 🏎️ RC Car Lap Timer — ESP32 + 光電式センサ

ESP32 と光電式センサを使ったラジコン用ラップタイマーです。
20×4 I2C LCD にリアルタイム表示し、BLE 経由でスマートフォンへラップタイムを送信します。

---

## 📷 システム概要

```text
[光電センサ] ──→ [ESP32] ──→ [I2C LCD 20×4]
                    │
                    └──→ [BLE] ──→ スマートフォン
```

センサがラジコンを検出すると自動でラップ計測を開始。
以降は通過ごとにラップタイムを記録・表示します。

---

## ✨ 機能

- **自動スタート** — 初回センサ通過で計測開始（手動操作不要）
- **リアルタイム表示** — 現在経過タイム / ベストラップ / 現在ラップ / 前ラップを LCD に表示
- **BLE 送信** — ラップ確定のたびにスマートフォンへ通知
- **デッドタイム制御** — 連続誤検出を防ぐ再トリガー禁止期間（ボリュームで 0.5〜5.0 秒に調整）
- **センサ感度調整** — 遮光判定時間をボリュームで 0〜200ms に調整（ノイズ除去）
- **設定確認モード** — スライドスイッチ切り替えで現在のパラメータを LCD に表示
- **EWMAフィルタ** — ADC 読み取りノイズをソフトウェアフィルタで平滑化

| 部品 | 仕様 |
| --- | --- |

| センサ | 光電式センサ（遮光型） |

### ピン配置

| GPIO | 用途 |
| --- | --- |
| 26 | 光電センサ入力（割り込み） |
| 32 | 感度調整ボリューム（ADC） |
| 33 | デッドタイム調整ボリューム（ADC） |
| 17 | LED 出力 |
| 14 | 設定モード切り替えスイッチ（INPUT_PULLUP） |
| 21 | I2C SDA（LCD） |
| 22 | I2C SCL（LCD） |

## 📺 LCD 表示レイアウト

### 計測待機中（初回センサ通過待ち）

```text
--- RC LAP TIMER ---
   LAP TIMER READY
    PASS SENSOR
      TO START
```

### 計測中

```text
TIME : 00:12.34    ^
BEST : 00:11.89
CURR : 00:12.34
PREV : 00:13.01   B+
```

| 表示 | 内容 |
| --- | --- |
| TIME | 現在ラップの経過時間（リアルタイム更新） |
| BEST | セッションベストラップ |
| CURR | 直前に確定したラップタイム |
| PREV | 2周前のラップタイム |
| `^` / `_` | センサ Ready（`^`）/ デッドタイム中（`_`） |
| `B+` / `B-` | BLE 接続中（`B+`）/ 未接続（`B-`） |

### 設定確認モード（スライドスイッチ切り替え時）

```text
--- SETTING MODE ---
SENSOR TH: 10.0 ms
DEAD TIME: 2.00 s
SENSOR   : CLEAR
```

## 📡 BLE 仕様

| 項目 | 値 |
| --- | --- |
| デバイス名 | `RC_LAPTIMER` |
| Service UUID | `12345678-1234-1234-1234-1234567890ab` |
| Characteristic UUID | `abcdefab-1234-5678-1234-abcdefabcdef` |
| プロパティ | READ / NOTIFY / INDICATE |
| 送信フォーマット | `Lap: 12.34`（秒単位） |

BLE スキャナアプリ（例：nRF Connect）で `RC_LAPTIMER` に接続すると、ラップ確定ごとに通知を受け取れます。
WEBクライアントも Pages で公開しています。[https://qoaopx.github.io/rc_laptimer/](https://qoaopx.github.io/rc_laptimer/)

## 🚀 使い方

1. 光電センサをコースのスタート/フィニッシュライン上に設置します
2. ESP32 に書き込んで電源を入れます（LED が 2 回点滅して起動確認）
3. LCD に `LAP TIMER READY / PASS SENSOR TO START` が表示されたら準備完了
4. ラジコンをセンサ前を通過させると計測開始
5. 以降、通過ごとに自動でラップタイムを記録・表示します

### ボリューム調整

| ボリューム | 調整内容 | 推奨値 |
| --- | --- | --- |
| 感度（GPIO32） | センサ遮光判定時間 0〜200ms | 5〜20ms |
| デッドタイム（GPIO33） | ラップ後の再検出禁止時間 0.5〜5.0s | コース 1 周の半分程度 |

## ⚙️ 技術的なポイント

- ISR 内の時刻取得に `esp_timer_get_time()`（μs 精度）を使用
- `volatile` 変数の読み取りを `noInterrupts()` で完全に保護
- ラップタイムを `unsigned long`（ms 単位）で保持し `float` による精度損失を回避
- EWMAフィルタ（α=0.1）で ADC ノイズを平滑化
- LED 制御・BLE 再接続をすべて非ブロッキングで実装
